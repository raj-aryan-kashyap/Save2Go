# 📍 InstaCompass

A lightweight, dark-mode mobile web application (PWA) designed for couples to seamlessly organize, track, and navigate their saved Instagram Reels and travel spots on the go.

## ✨ Key Features
* 🔄 **Real-Time Google Sheets Integration:** Acts as a zero-cost, live cloud database that updates instantly for both users.
* 👥 **Dual-User Collaboration & History Tracking:** Custom sessions for two users with an automated audit log tracking who marked spots as done or starred.
* 📍 **Smart Proximity Sorting:** Uses browser-based HTML5 Geolocation to automatically surface the closest spots to your current position.
* 🔍 **Offline Caching:** Saves database backups to local device storage, allowing quick reading of notes and directions even without cellular service.
* 📄 **Ticket Dashboard:** Links specific booked venues directly to ticket PDFs stored in Google Drive.
